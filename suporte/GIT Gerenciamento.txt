Alterar o conteúdo de home.py

# 1) ir para a main e garantir que está atualizada
git checkout main
git pull origin main

# 2) trazer as mudanças da dev
git merge dev -m "release: promove dev para main"

# (se houver conflitos, resolva no VS Code, git add ., e git commit)

# 3) publicar na main (dispara o deploy)
git push origin main

dev para ia
------------
# Certifique-se de estar na branch `dev`
git checkout dev

# Atualize a branch `dev` com as alterações da branch `ia`
git merge ia

# Resolva quaisquer conflitos, se houver, e finalize o merge
# Após resolver os conflitos:
git add .
git commit -m "Resolve conflitos e atualiza a branch dev com as alterações da branch ia"

# Opcional: envie as alterações para o repositório remoto
git push origin dev
